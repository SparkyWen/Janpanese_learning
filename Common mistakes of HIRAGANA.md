# Common mistakes

## 0）通用“快速辨认法”（先记这个会越学越省力）

看一个假名时，先按顺序扫 4 个锚点：

1. **横线有几条？**（0 / 1 / 2）
2. **竖线是 1 根还是 2 根？**
3. **有没有“圈/回环”？圈在上/中/下？**
4. **尾巴落点**：最后一笔是往**右下**、还是往**左下**、还是收在中间？

> 你会发现：你混淆的那些字，几乎都能用“横线数 + 竖线数 + 圈的位置 + 尾巴落点”秒分。

------

## 1）あ (a) vs お (o) ——“有叉/没叉、有大圈/没大圈”

### 一眼锚点

- **あ**：中间更像有个“小交叉/骨架”（你能感觉到“结构更硬”）。
- **お**：特征是**右边一个更明显的“大回环/大圈”**，整体更“圆”。

### 记忆口诀

- **あ = 结构感强（像有骨架）**
- **お = 圈更大（o 像 big O）**

### 练习法（超有效）

- 每次写 **お** 都强迫自己把右侧圈写大一点；写 **あ** 就把“中间的交叉感”写出来。
- 交替写：あ お あ お あ お（每个都读出来：a o a o…）

------

## 2）け (ke) vs は (ha) vs ほ (ho) ——“两竖是け；ほ是は+一小横”

### 一眼锚点

- **け**：最关键是**像“2 根竖”并排**（整体偏直、偏“柱子感”）。
- **は**：整体像“左边一小竖 + 右边一大弯（带回环）”。
- **ほ**：就是 **は 的形状 + 多了一笔小横（小短线）**（你截图里那条小短线很明显）。

### 记忆口诀

- **け：看“竖竖”（两根竖）**
- **ほ = は + 一横**（“ho 多一横”）

### 练习法

- 先写一排：けけけ（只练“两竖感”）
- 再写：は は は（只练“右边大弯+回环”）
- 最后写：は → ほ（在“は”上加那条小横），让大脑形成“加一笔就变 ho”。

------

## 3）ま (ma) vs も (mo) ——“も有两横；ま像有个‘门/口’的拐弯”

### 一眼锚点

- **も**：最稳的特征是**两条横线**（上面一横 + 中间一横），然后再往下带尾巴。
- **ま**：通常是**上面一横**，下面更像“拐进去/绕一下”（更像有个“角/门框”的感觉），而不是“再来一条中横”。

### 记忆口诀

- **も：‘m o’——o 圆、所以中间再“补一横”让它更饱满（两横）**
- **ま：只有一横，下面像“绕进门”**

### 练习法

- 写 **も** 时先口头说：“上横 + 中横 + 尾巴”。
- 写 **ま** 时口头说：“上横 + 往下绕一下（不要写中横）”。

------

## 4）た (ta) vs な (na) ——“た像有‘十字/顶横压住’；な更圆更绕”

### 一眼锚点

- **た**：上方更容易看成“**顶横压住下面**”（整体更像有“十字感/支架感”）。
- **な**：整体更“**顺滑、绕出来**”，没有那么强的“顶横压住”的感觉。

### 记忆口诀

- **た：像‘T’（ta 里有 t），上面压一下**
- **な：更软、更绕（n 像弯）**

### 练习法

- 把 **た** 的“顶横”写得稍微更坚定、更平一点；
- 把 **な** 写得更圆滑（尤其下面那一段不要写出“十字”的硬感）。

------

## 5）ち (chi) vs ら (ra) ——“ち有更明显的‘上横+长尾’；ら像一笔顺下来的弯”

### 一眼锚点

- **ち**：你要抓住**上面那条更明显的横**，以及**下面通常会拖出比较清楚的尾巴**（整体像“上面定一下，再往下走”）。
- **ら**：整体更像**一笔顺下来形成的弯**，上面的“定一下”没那么强（更像自然下落的弧线）。

### 记忆口诀

- **ち：先“停一下”（上横），再往下拖尾**
- **ら：顺下来一笔弯（像 ra 的 r：rolling 下去）**

### 练习法

- 写 ち 时刻意做一个“停顿感”：上横写完，心里停 0.2 秒再下去。
- 写 ら 时尽量“不断笔/不断气”地顺下来。

------

## 6）り / る / れ / ろ ——你圈的这一组最常见：用“是否两笔、是否有圈、是否有左边竖”分

### 一眼锚点（按最稳的特征排）

- **り (ri)**：**两根分开的竖线感**（两笔、像“||”的气质）。
- **る (ru)**：中间有**明显的小回环/小圈**，而且整体更圆。
- **れ (re)**：很关键：**左边像有一根“竖/立起来的起笔”**，后面再带弯尾（比 る 更“有左边那根立着的感觉”）。
- **ろ (ro)**：更像一个**圆角的“口/框”**（整体更封闭、更“围起来”），没有 る 那种“中间小圈+甩尾”的感觉。

### 记忆口诀

- **り：两竖 = ri**
- **る：有圈 = ru**（“ru 有 round 的圈”）
- **れ：左边先立一下 = re**
- **ろ：像个圆口 = ro（口=口）**

### 练习法（推荐你这么练）

做“最小对立”练习：

- り vs る：**两竖** vs **一圈**
- る vs ろ：**圈在里面+甩尾** vs **像个框更封闭**
- れ vs る：**れ 左边先立**，る 没那么“立”

------

## 7）わ / ね / ぬ ——“三兄弟”：都像“绕一下+尾巴”，用“有没有明显的结/圈、尾巴走势”分

> 你说它们“很相似”非常正常，这组需要靠“圈的位置 + 尾巴”记。

### 一眼锚点（把它们当成“同一类字”的不同尾巴）

- **わ (wa)**：整体更“简单”，**绕一下就走**，没有太复杂的“结”。（你图里わ看起来最清爽）
- **ね (ne)**：通常会有一种“**打结/绕回来的感觉**”，像“绕了一下又折回来再出去”。
- **ぬ (nu)**：最像“**绕出一个更圆的圈**”，然后尾巴拖出去（很多人觉得它像在画一个“绳结+长尾”）。

### 记忆口诀（很实用）

- **わ：one loop 就走（最简单）**
- **ね：像打了个“结”（ne = knot）**
- **ぬ：圈更圆、更像“nu=loop 更明显”**

### 练习法

- 先写 3 个“同样大小的圈”，分别变形为 わ / ね / ぬ：
  - わ：圈小一点、尾巴干净
  - ね：中间做“回折感”
  - ぬ：圈更圆、尾巴更明显
- 每次写完马上读：wa / ne / nu（让视觉和发音绑定）

------

## 8）顺便把你学到的 Dakuon / Handakuon 也“绑定”到记忆里（不再额外背形状）

你已经在学浊音/半浊音了，这里记一个最省力的规则：

- **浊音 Dakuten：゛**（两点）= “加了浊音，不改字形主体”
  - か→が、さ→ざ、た→だ、は→ば（主体不变，只加两点）
- **半浊音 Handakuon：゜**（小圆圈）= **只用在 h 行**
  - は→ぱ、ひ→ぴ、ふ→ぷ、へ→ぺ、ほ→ぽ（只给 h 行加小圈）

> 也就是说：你把 **は/ほ** 这组写熟了，再加两个点或一个圈，就顺带掌握 **ば/ぼ、ぱ/ぽ**，性价比非常高。
