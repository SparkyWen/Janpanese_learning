# Dakuon / Handakuon

你这张图这一块叫 **浊音（だくおん）** 和 **半浊音（はんだくおん）**。核心其实只有一句话：

> **看“原来的五十音是哪一行”，再加符号就变成对应的 g / z / d / b / p。**

下面我用“你已经会五十音”的前提，给你一个最稳的区分方法 + 记忆法。

------

## 1）先记一个总规则：两点=“变浊”（带嗓音），小圈=“变 p”

- **゛两点（浊点）**：让辅音“带嗓音震动”（voiced）
  - か行 → が行（k→g）
  - さ行 → ざ行（s→z / j）
  - た行 → だ行（t→d / j / z）
  - は行 → ば行（h→b）
- **゜小圈（半浊点）**：只作用在 **は行**，变成 **ぱ行（p）**

你可以用一个很短的口令记：

> **K→G，S→Z，T→D，H→B / P**

------

## 2）如何“瞬间判断到底是 g / z / d / b / p”？

### ✅ 方法：先把符号“擦掉”，看它原本是谁

比如你看到：**ぎ**

- 擦掉两点 → **き**
- き 是 **か行**
- か行 + ゛ ⇒ **が行** ⇒ 所以它读 **gi（ぎ）**

同理：

- **ぜ**：擦掉两点→せ（さ行）→ざ行→**ze**
- **ど**：擦掉两点→と（た行）→だ行→**do**
- **ぼ**：擦掉两点→ほ（は行）→ば行→**bo**
- **ぽ**：擦掉圈→ほ（は行）→ぱ行→**po**

所以你根本不需要“凭感觉猜 g/z/d/b/p”，只要追溯它原来的五十音行就行。

------

## 3）每一组你应该怎么记（把它当成“同一行的升级版”）

### (1) か行 + ゛ → が行（g）

**かきくけこ → がぎぐげご**
 记忆点：**k 变成带嗓音的 g**
 练习：

- か(ka) ↔ が(ga)
- き(ki) ↔ ぎ(gi) ……

### (2) さ行 + ゛ → ざ行（z / j）

**さしすせそ → ざじずぜぞ**
 这里唯一“看起来烦”的是：**し + ゛ = じ（ji）**
 记忆点：

- **ざ(za) じ(ji) ず(zu) ぜ(ze) ぞ(zo)**

### (3) た行 + ゛ → だ行（d / j / z）

**たちつてと → だぢづでど**
 这里又会出现：

- ち + ゛ = **ぢ（ji）**（听起来像 ji）
- つ + ゛ = **づ（zu）**（听起来像 zu）

**你问“怎么区分 z 和 d？”**
 答案是：**看它原本是 さ行还是 た行。**

- 原本是 し/す/せ/そ/さ ⇒ ざ行（ざじずぜぞ）
- 原本是 ち/つ/て/と/た ⇒ だ行（だぢづでど）

> 也就是说：你不是在区分“声音像不像”，而是在区分“它原来是哪一行”。

### (4) は行 + ゛ → ば行（b）

**はひふへほ → ばびぶべぼ**

### (5) は行 + ゜ → ぱ行（p）

**はひふへほ → ぱぴぷぺぽ**
 记忆点：小圈像“泡泡/气”，p 是更“喷气”的音。

------

## 4）一个你必须知道的现实：じ vs ぢ、ず vs づ —— 现代日语里大多读音一样

很多地区/场景里：

- **じ** 和 **ぢ** 都听起来像 **ji**
- **ず** 和 **づ** 都听起来像 **zu**

所以学习时容易困惑。最实用的规则是：

### ✅ 拼写规则（不要靠听）

- **ざ行**：ざ・**じ**・ず・ぜ・ぞ（来源：さ行）
- **だ行**：だ・**ぢ**・づ・で・ど（来源：た行）

常见用法小提示（知道就够，不用死背太多例外）：

- **ぢ/づ** 多出现在“词根原本是 ち/つ”的词里，比如：
  - ちぢむ（缩）
  - つづく（继续）
  - はなぢ（鼻血）
- 其他绝大多数情况你会更多见到 **じ/ず**。

------

## 5）发音记忆：用“摸喉咙法”让大脑建立肌肉记忆

把手轻放喉咙（声带位置）：

- k / s / t / h：**不怎么震动**
- g / z / d / b：**明显震动（浊音）**
- p：不震动，但气更冲（爆破感更强）

这对你区分“浊音的感觉”特别有效。
